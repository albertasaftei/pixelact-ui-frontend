import { PixelButton } from "pixelact-ui";

# PixelButton

A pixel art styled button component built on top of shadcn/ui's Button component. Features retro styling with multiple variants and full support for all underlying button props.

## Usage

### Basic Button

<CodePreview 
  language="tsx"
  code={`import { PixelButton } from "pixelact-ui";

export function BasicButton() {
  return (
    <PixelButton>
      Click Me!
    </PixelButton>
  );
}`}
>
  <PixelButton>Click Me!</PixelButton>
</CodePreview>

### As Child (Polymorphic)

```tsx
import { PixelButton } from "pixelact-ui";
import { Link } from "react-router-dom";

export function AsChildButton() {
  return (
    <PixelButton asChild>
      <Link to="/dashboard">Go to Dashboard</Link>
    </PixelButton>
  );
}
```
