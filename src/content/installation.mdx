<p className="text-2xl pixel-font">Get started with <br /> Pixelact UI</p>

<p className='mb-2 underline pixel-font'>Installation</p>
<p>
  To install <strong>Pixelact UI</strong>, you can use npm.
</p>
<div className='mt-2'>
```bash 
npm install pixelact-ui
```
</div>
<br />

<p>
  After installing, <strong>import the CSS</strong> in your main CSS file. This
  is required in order to apply the default styles to the
  components.
</p>

<div className="mt-2">
```jsx
import '@pixelact-ui/styles.css';
```
</div>
<br />

<p className="mb-2 underline pixel-font">Usage</p>
<p>
  To use <strong>Pixelact UI</strong> components, import them into your code
  like this:
</p>

<div className="mt-2">
```jsx
import { PixelButton } from '@pixelact-ui/react';

function MyComponent() {
  return (
    <PixelButton variant="primary">Click me</PixelButton>
  );
}
```
</div>

<br />

<p className="mb-2 underline pixel-font">Styling</p>
<p>
  Pixelact UI components are built on top of{" "}
  <span className="underline">shadcn/ui</span> components. Every customization
  that can be done on shadcn/ui components can be done on Pixelact UI components
  as well using Tailwind CSS.
</p>
<br />

<p className="mb-2 underline pixel-font">Theme support</p>
<p>
  The components are designed to work with both light and dark themes. You can
  easily override the styles changing the CSS variables in the root element of
  the main CSS file.
</p>

</div>
